# Aditya Sharma Portfolio

Static portfolio website for Aditya Sharma, focused on machine learning, computer vision, Android privacy tooling, and edge AI projects.

## Contents

- `index.html` - main portfolio page
- `styles.css` - responsive styling
- `script.js` - interactive resume key panel
- `assets/resume/aditya-sharma-resume.pdf` - downloadable resume
- `assets/images/resume-preview.png` - visual resume preview used on the homepage

Open `index.html` in a browser to view the site locally.
